# IoT Project

## 📌 Project Description
This is a PHP + MySQL based web project developed using XAMPP.

## ⚙️ Technologies Used
- PHP
- MySQL
- HTML, CSS, JavaScript
- XAMPP Server

## 🚀 How to Run
1. Install XAMPP
2. Copy project to `htdocs` folder
3. Start Apache and MySQL
4. Import database in phpMyAdmin
5. Open browser:
   http://localhost/iot

## 🗄️ Database Setup
- Open phpMyAdmin
- Create database
- Import `database.sql` file
